# html-links-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What HTML tag is used to link to another website?
  anchor <a> element

- What is an absolute URL?
  points to a location defined by its absolute location on the web (includes protocol and domain name)
  always points to the same location no matter where it's used

- What is a relative URL?
  points to a location that is relative to the file you are linking from
